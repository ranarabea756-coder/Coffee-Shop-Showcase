# Coffee Shop Website

A modern coffee shop website built using HTML and CSS, featuring a clean layout, interactive sections, and elegant visual styling inspired by contemporary café brands.

## Overview

This project presents a complete coffee shop landing page with multiple sections designed to create a smooth and engaging user experience.

## Features

* Modern responsive-style layout
* Interactive navigation bar with dropdown menu
* Product showcase sections
* Trending drinks and toppings display
* Customer review section
* Blog and latest collection cards
* Animated hover effects and transitions
* Newsletter subscription footer

## Technologies Used

* HTML5
* CSS3
* Font Awesome

## Design Focus

The project focuses on:

* Clean UI design
* Structured page layout
* Smooth animations and hover interactions
* Modern coffee shop branding aesthetics

## Purpose

This project was created to practice front-end web development skills and improve experience with layout design, animations, and user interface styling using pure HTML and CSS.
